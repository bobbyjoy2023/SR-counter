# Iron Streak — Build Instructions

This repo builds an installable Android APK automatically using GitHub Actions.

## Get your APK (no PC software needed)

1. Create a free account at github.com (if you don't have one)
2. Create a new repository (private is fine)
3. Upload ALL files/folders from this zip to the repo (keep the folder structure,
   including the hidden `.github` folder)
4. Go to the **Actions** tab — the "Build APK" workflow starts automatically
5. When it finishes (~3-5 min), click the run → download the **IronStreak-APK** artifact
6. Unzip it → you get `app-debug.apk`
7. Transfer to your phone (or do all of the above from your phone's browser)
8. On the phone: tap the APK → allow "Install unknown apps" for your browser/Files app → Install

## App features
- Start date/time picker → live streak timer
- Milestone celebrations at days 7, 14, 21, 30, 60, 90 with motivational quotes
- Stacking reward medallions
- Daily rotating motivation on open
- Black & red theme, fully offline, data stored on-device
